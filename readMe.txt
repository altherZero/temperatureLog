---------------------------------------------Arduino ESP322--------------------------------------------

1. instalar Arduino IDE
2. gestor de tarjetas buscar e instalar paquete de esp32 espressif systems 
3. admiistrador de librerias instalar buscar e instalar max6675 de adafruit
4. instalar driver esp32 https://www.silabs.com/developer-tools/usb-to-uart-bridge-vcp-drivers?tab=downloads
5. conectar usb y verificar puerto COM#
